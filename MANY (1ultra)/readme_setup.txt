PASOS NIVEL FACIL:
1. Subir estos archivos a tu GitHub (nuevo repo).
2. En Render: New Web Service -> conecta con tu GitHub.
3. Selecciona MANY-Bot -> Deploy.
4. Variables de entorno: ADMIN_USER, ADMIN_PASS, API_KEY.
5. Abrir la URL que Render te da -> /login.
6. Entrar con admin / 12345Segura (o lo que pongas en .env).
